# MathGrade12: Total Skillz Platform

> **Built To Forge Greatness.**

Welcome to **MathGrade12**, a premium web-based educational platform designed to help students master Grade 12 Mathematics through interactive practice, data-driven insights, and high-end design.

---

## Features

- **Phone Number Authentication**: Secure and accessible sign-in via SMS verification (OTP).
- **Interactive Practice**: Dynamic problem sets that challenge your skills and provide instant feedback.
- **Performance Dashboard**: Track your progress and identify areas for improvement with a sleek data visualization interface.
- **Curated Resources**: Access cheat sheets and practice papers (NSC) directly from the platform.
- **3D Immersive Design**: A modern, high-contrast user interface featuring Three.js animations for a premium learning experience.
- **Secure Implementation**: Built with security best practices, including HTTP headers protection and rate limiting.

---

## Tech Stack

### Frontend
- **HTML5 & CSS3**: Custom responsive layouts with premium glassmorphism effects.
- **JavaScript (Vanilla)**: Core application logic and interactive components.
- **Three.js**: Immersive 3D motion and visual effects.
- **Google Fonts**: Typographic excellence using modern typefaces.

### Backend & Cloud
- **Firebase Authentication**: Secure user login and Google Sign-In.
- **Cloud Firestore**: Real-time NoSQL database for user progress persistence.
- **Firebase Hosting**: Fast and secure global content delivery.
- **Node.js & Express.js**: Server-side logic and security middleware.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- npm (installed automatically with Node.js)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sphe-hlongwa/Projects.git
   cd MathGrade12
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the server**:
   ```bash
   npm start
   ```

4. **View the app**:
   Open your browser and navigate to `http://localhost:3000`.

## Project Structure

- `/public`: Frontend assets (HTML, CSS, JS, Images).
- `/problems`: PDF resources and exam papers.
- `server.js`: Express server configuration and security middleware.
- `netlify.toml`: Deployment configuration for Netlify.
